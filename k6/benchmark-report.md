## Implementation

### Before (21 queries) - N+1 original
```java
String name = memberNamePort.getNameByMemberId(post.getAuthorId()); // SELECT x N
int cnt = commentRepository.countByPostId(post.getId());            // COUNT  x N
// 10 posts per page -> 1 + 10 + 10 = 21 queries
```

### Method2 (3 queries) - Batch IN + Map
```java
Map<Long,String> nameMap = memberNamePort.getNamesByMemberIds(authorIds); // 1 IN query
Map<Long,Long>   cntMap  = commentRepository.countsByPostIds(postIds);    // 1 GROUP BY
// 10 posts -> 1(list) + 1(IN) + 1(GROUP BY) = 3 queries
```

### Method3 (1 query) - JOIN + DTO Projection
```sql
SELECT p.id, p.board_type, p.title, m.name, p.created_at, p.view_count, COUNT(c.id)
FROM posts p
JOIN members m ON m.id = p.author_id
LEFT JOIN comments c ON c.post_id = p.id
WHERE p.board_type = ? AND p.status = 'ACTIVE'
GROUP BY p.id, p.board_type, p.title, m.name, p.created_at, p.view_count
ORDER BY COUNT(c.id) DESC
```
